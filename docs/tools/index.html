    # 系统

    • Kali Linux - 网络安全最常用的Linux系统，内置了几乎所有常用Linux工具

    # 软件

    • Firefox (opens new window) - 常用的浏览器，开发者工具功能很多(全平台)
    • Hackbar (opens new window) - 浏览器拓展Web安全调试工具(全平台)
    • Fiddler (opens new window) - 常用的抓包工具(Windows，Linux/Mac上面有Fiddler Everywhere需要连接外网使用)
    • Chrome (opens new window) - 开发者常用的浏览器(全平台)
