# About BUUCTF

BUUCTF是北京联合大学智慧城市学院和安恒信息联合开设的CTF在线评测站，题目大多数来源于CTFTraining项目 (opens new window)。

  • 访问速度：很快。
  • 是否要求登录：是。
  • 收费情况：免费，无金币限制。
