# Web7 浏览器JS调试

这道题主要考察对JS调试器的使用。

打开环境，可以看到页面在不停的刷新，根据题目要求，在Panda处停下就可以得到flag。在「开发中工具-调试器」中，先将页面暂停。

因为图片是随机载入的，所以我们点击步出（Shift+F11），一步一停，观察flag是否出现。

之后就可以截到Flag了。
